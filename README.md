# This game is Tic Tac Toe but hexagonal
## Idea took from webgoatguy: https://youtu.be/Ob6QINTMIOA
## I really recommend watching his video to fully understand the game

# Rules:
- First turn has only 1 move
- Each player has 2 moves after the first turn
- First to make a row of 6 tiles wins
